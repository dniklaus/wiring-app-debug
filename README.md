# wiring-app-debug
Application debug setup component for Wiring / Arduino framework based projects.

Dependencies:

* SerialCommand (<https://github.com/kroimon/Arduino-SerialCommand>)
* DbgTrace (1.1.0) (<https://github.com/ERNICommunity/dbg-trace>)
* DebugCli (1.3.0) (<https://github.com/ERNICommunity/debug-cli>)
* RamUtils (2.1.0) (https://github.com/dniklaus/arduino-utils-mem)
* SpinTimer (3.0.0) (<https://github.com/dniklaus/spin-timer>)

Usage:
For demonstration how to use this component, check out the following projects:

* wiring-skeleton (<https://github.com/ERNICommunity/wiring-skeleton>)
* wiring-iot-skeleton (https://github.com/ERNICommunity/wiring-iot-skeleton)
* wiring-lora-skeleton (https://github.com/ERNICommunity/wiring-lora-skeleton)
